API Overview

Placeholder Image Text (PIT) is a free, open-source API to generate custom placeholder images with text, neon effects, gradients, borders, and more — all via a simple URL.

⚡ Base URL: https://your-domain.com  ·  No auth required  ·  CORS enabled
💡 Tip: Images are generated on-the-fly. Cache-Control is set to 24h. Max size: 4096×4096px.
GET /api — Health check & info
Returns API status, version, and available endpoints as JSON.

Endpoints

All endpoints return a PNG or JPEG image. Mix and match parameters freely.

GET /{width}x{height}
Main image generation endpoint. Example: /600x400?text=Hello&bg=0d0d0d
GET /api/{width}x{height}
API-prefixed route. Identical to above. Example: /api/600x400?text=Hello
GET /{width}x{height}/{bg-color}
Path-based color. Example: /600x400/dark or /600x400/7209b7
GET /{width}x{height}/neon=true/{bg}?text-colour={color}&text={text}
Supports your original URL format from the path. Example: /600x400/neon=true/dark?text-colour=white&text=Hello

Query Parameters

All parameters are optional. Parameters can be passed as query strings or in the URL path.

Named Colors

Use these names anywhere a color is expected (bg, text-colour, border-color, gradient).

Examples for every major language and framework. Replace https://your-domain.com with your actual domain.

JavaScript — Fetch API

JAVASCRIPT
// Basic usage — insert image into DOM
const img = document.createElement('img');
img.src = 'https://your-domain.com/600x400?bg=0d0d0d&text-colour=white&text=Hello+World&neon=true';
document.body.appendChild(img);

// Download image via Fetch
async function downloadPlaceholder() {
  const url = `https://your-domain.com/800x400?bg=7209b7&text-colour=c084fc&text=Neon+Purple&neon=true&bold=true`;
  const res = await fetch(url);
  const blob = await res.blob();
  const link = document.createElement('a');
  link.href = URL.createObjectURL(blob);
  link.download = 'placeholder.png';
  link.click();
}

// Build URL dynamically
function buildImageUrl(w, h, opts = {}) {
  const base = 'https://your-domain.com';
  const params = new URLSearchParams(opts);
  return `${base}/${w}x${h}?${params}`;
}
console.log(buildImageUrl(600, 400, { bg: '0d0d0d', 'text-colour': 'white', text: 'Hello', neon: 'true' }));

Node.js

NODE.JS (CJS)
const https = require('https');
const fs = require('fs');

// Download image to file
function downloadImage(url, filepath) {
  return new Promise((resolve, reject) => {
    https.get(url, (res) => {
      const file = fs.createWriteStream(filepath);
      res.pipe(file);
      file.on('finish', resolve);
      file.on('error', reject);
    });
  });
}

const url = 'https://your-domain.com/600x400?bg=1a1a2e&text-colour=c084fc&text=Node.js+Image&neon=true';
downloadImage(url, 'output.png').then(() => console.log('Saved!'));

fs.writeFileSync('og-image.png', buffer);

Python

PYTHON 3
import requests
from urllib.parse import urlencode

BASE_URL = "https://your-domain.com"

# Simple download
def generate_image(width, height, **kwargs):
    params = urlencode(kwargs)
    url = f"{BASE_URL}/{width}x{height}?{params}"
    response = requests.get(url)
    return response.content

# Generate and save
img_bytes = generate_image(
    600, 400,
    bg="0d0d0d",
    **{"text-colour": "22d3ee"},
    text="Python+Image",
    neon="true",
    bold="true"
)
with open("output.png", "wb") as f:
    f.write(img_bytes)
print("Image saved!")

# Display in Jupyter/IPython
from IPython.display import Image, display
url = f"{BASE_URL}/800x400?bg=7209b7&text-colour=white&text=Jupyter+Preview&neon=true"
display(Image(url=url))

React / Next.js

JSX / TSX
// PlaceholderImage component
function PlaceholderImage({ width = 600, height = 400, text, bg = '0d0d0d', color = 'white', neon = false }) {
  const params = new URLSearchParams({ bg, 'text-colour': color, ...(text && { text }), ...(neon && { neon: 'true' }) });
  const src = `https://your-domain.com/${width}x${height}?${params}`;
  return <img src={src} width={width} height={height} alt={text || `${width}x${height}`} />;
}

// Usage
export default function Page() {
  return (
    <div>
      <PlaceholderImage width={800} height={400} text="Hero Image" bg="07070f" color="c084fc" neon />
      <PlaceholderImage width={300} height={300} text="Avatar" bg="7c3aed" color="white" />
    </div>
  );
}

Vue.js

VUE 3
<template>
  <img :src="imageUrl" :width="width" :height="height" :alt="text" />
</template>

<script setup>
import { computed } from 'vue';

const props = defineProps({
  width: { default: 600 }, height: { default: 400 },
  text: String, bg: { default: '0d0d0d' }, color: { default: 'white' }, neon: Boolean
});

const imageUrl = computed(() => {
  const p = new URLSearchParams({ bg: props.bg, 'text-colour': props.color });
  if (props.text) p.set('text', props.text);
  if (props.neon) p.set('neon', 'true');
  return `https://your-domain.com/${props.width}x${props.height}?${p}`;
});
</script>

Deploy Guide

Deploy this API to any Node.js-compatible platform. The project is a standard Express.js server using canvas for image generation.

⚠️ The canvas package requires native binaries. Platforms that support Node.js with native modules are recommended (Vercel, Railway, Render, VPS).

Vercel

vercel.json
{
  "version": 2,
  "builds": [{ "src": "server.js", "use": "@vercel/node" }],
  "routes": [{ "src": "/(.*)", "dest": "server.js" }]
}
TERMINAL
npm i -g vercel
vercel login
vercel --prod
